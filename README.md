# 📊 SaaS Cohort Retention & Customer Lifetime Value (CLTV) Analysis

A professional Data Analytics project focused on understanding customer retention patterns and estimating Customer Lifetime Value (CLTV) using transactional data. This project applies data cleaning, exploratory data analysis, cohort analysis, customer segmentation, and business intelligence techniques to generate actionable insights for SaaS and E-commerce businesses.

---

## 📖 Project Overview

Customer retention is one of the most important metrics for subscription-based and e-commerce businesses. Acquiring new customers is expensive, making it essential to understand how long customers stay, how often they purchase, and how much value they generate over time.

This project performs:

- Customer Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Cohort Retention Analysis
- Customer Lifetime Value (CLTV) Calculation
- Customer Segmentation
- Business Insights & Recommendations
- Interactive Dashboard Development

---

## 🎯 Objectives

- Clean and preprocess transactional data
- Analyze customer retention across cohorts
- Calculate Customer Lifetime Value (CLTV)
- Identify high-value customer segments
- Visualize retention trends and revenue insights
- Provide data-driven business recommendations
